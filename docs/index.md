# AETHER-PULSE // OMNI-LOOP

Welcome to the documentation for **AETHER-PULSE**, a localized, high-assurance autonomic defense engine designed for edge execution.

## The Paradigm
The system unifies real-time kernel telemetry, WebAssembly-based instant remediation reflexes, and a localized NPU-accelerated Large Language Model (Gemma 2) to dynamically identify and neutralize sophisticated threats without external dependencies.

## Quick Start
To spin up the cognitive core:
```bash
docker compose up -d
```
